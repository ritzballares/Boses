# Boses
Boses is an iOS app that aims to help people learn ASL fingerspelling. This project was presented at Howard Community College's 2021 Scholars Symposium.

## Demo
![Image](./boses_demo.gif)

## Technologies used
- Swift
- UIKit
-  Create ML
- Core ML
